# Conversation Settings Page

The **Conversation Settings Page** is the UI configuration panel for individual conversation sessions, located in `frontend/src/pages/ConversationSettingsPage.tsx`.

## Features
- **Connection Configuration**: Allows users to configure custom endpoint profiles (Base URL, API Key, Model, and API Type) for a specific conversation session.
- **Model Overrides**: Overrides global admin settings, letting users assign different models (e.g. Qwen, Claude, GPT) to different tabs.
- **System Prompt Editing**: Provides custom system prompt textareas, allowing users to specialize the behavior of the agent for custom tasks.
