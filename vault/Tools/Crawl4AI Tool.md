# Crawl4AI Tool

`Crawl4AITool` (defined in `app/tool/crawl4ai.py`) utilizes the crawl4ai library to parse, extract, and clean contents of web pages.

## Operations
- **Smart Chunking**: Extracts main article bodies, tables, and paragraphs while ignoring sidebars, banners, and scripts.
- **Output Formats**: Outputs formatted Markdown or JSON blocks, making it easy for the agent to ingest and analyze web data.
