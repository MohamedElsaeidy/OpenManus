# Skill Playbook Tool

`SkillPlaybook` (defined in `app/tool/skill_playbook.py`) is a tool that allows the agent to execute reusable, complex workflows called skills.

## Details
- **Verification Commands**: Encourages developers to run the narrowest meaningful verification command first, followed by broader checks if shared behavior changes.
- **Workflow Automation**: Automates complex workspace tasks by querying predefined script collections, speeding up the implementation-verification loop.
