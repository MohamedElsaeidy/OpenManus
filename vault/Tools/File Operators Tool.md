# File Operators Tool

`FileOperators` (defined in `app/tool/file_operators.py`) gathers basic filesystem operations under a single tool interface.

## Actions
- **Create**: Writes new files to specified paths.
- **Read**: Reads full file contents as string blocks.
- **Write**: Overwrites target files atomically.
- **List & Search**: Scans local directories for file items.
