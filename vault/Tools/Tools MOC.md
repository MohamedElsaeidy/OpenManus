# Tools MOC

This Map of Content (MOC) indexes the tools available to OpenManus agents. These tools allow agents to interact with the local workspace, control web browsers, execute terminal commands, edit source files, and prompt users for input.

## Code Editing & Terminal Execution
- [[Line Edit Tool]] — Fast, line-number-based editor.
- [[Str Replace Editor]] — String-matching file editor.
- [[Apply Patch Editor]] — Atomic multi-file unified diff editor.
- [[Bash Tool]] — Shell command runner inside sandbox environments.
- [[Python Execute Tool]] — Interactive Python interpreter sandbox.

## Web & Codebase Retrieval
- [[Browser Use Tool]] — Playwright/browser-use web automation.
- [[Web Search Tool]] — Google and DuckDuckGo search integration.
- [[Codebase Tool]] — Work space overview and ripgrep grep capabilities.

## Execution Control & State
- [[Planning Tool]] — Task state, task lists, and milestone management.
- [[Terminate Tool]] — Structural finish handler for agent loops.
- [[Ask Human Tool]] — Human-in-the-loop validation prompt.
