# Computer Use Tool

`ComputerUseTool` (defined in `app/tool/computer_use_tool.py`) provides OS-level interactions such as keyboard typing, mouse navigation, and screen captures.

## Design
- **Action Schema**: Models actions like mouse clicks, mouse drags, key presses, and scrolls on coordinates.
- **Visual Feedback**: Generates screen captures and parses window geometries, allowing visual agents to operate GUI-based software.
