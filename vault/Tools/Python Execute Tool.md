# Python Execute Tool

`PythonExecute` (defined in `app/tool/python_execute.py`) is an interactive Python runtime environment tool.

## Key Properties
- **Execution Sandbox**: Runs Python code snippets inside isolated environments to prevent execution impacts on the host machine.
- **State Preservation**: Retains variables, functions, and import environments between consecutive calls to allow step-by-step code execution.
