# Home

Welcome to the **OpenManus codebase Obsidian Vault**. This vault represents a structured, navigable map of the OpenManus system, covering the core agent logic, server APIs, tools, user interfaces, deployment configuration, and known system issues.

Use the Map of Content (MOC) links below to navigate the different modules of the codebase:

- [[Agent Core MOC]] — The central agent schemas, base classes, execution loops, and specialized agent personas.
- [[Tools MOC]] — Local and remote (MCP) tools available to the agents for executing workspace commands and external integrations.
- [[Server MOC]] — Fast API endpoints, Celery workers, background tasks, and Obsidian synchronization handlers.
- [[Frontend MOC]] — React-based web interface, chat step timeline visualization, and settings pages.
- [[Deployment MOC]] — Docker configurations, multi-container stacks, and workspace volume mounts.
- [[Known Issues MOC]] — Seeded security risks, architectural bottlenecks, and synchronization bugs in the current implementation.
